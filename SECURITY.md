# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 1.0.x | Yes |

## Reporting a Vulnerability

If you find a security issue, please avoid filing a public issue before disclosure is coordinated.

Send a private report including:

- affected repository and platform
- steps to reproduce
- impact summary
- screenshots, logs or proof of concept when relevant

Contact:

- Email: `juliocesarmoralesalvarado9@gmail.com`

Initial response target: within 72 hours.
When possible, fixes will be prepared privately and disclosed after a patch is available.
